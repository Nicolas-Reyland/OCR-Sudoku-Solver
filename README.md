# OCR-Sudoku-Solver
OCR Sudoku Solver Projet S3 Epita. Here is a [link to the project's website](https://reyland.dev/sudoku).

## Installation
You need to have these installed :
 * libsdl2-dev
 * libsdl2-ttf-dev
 * libgtk-3-dev

To compile our project :
```bash
make
```

## Usage
To run our project :
```bash
./main
```

## Test suite
To run the test suite :
```bash
make test
```
